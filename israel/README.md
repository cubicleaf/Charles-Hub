# Israel Tracker

Israel/Palestine/Mideast tracker — violence, legal machinations, key-figure rhetoric, source-faithful summaries with encyclopedic background context.

Replaces the unused `news/` subject. Git history for the folder is preserved via `git mv`.

---

## Scope

- **Physical violence** — IDF operations, settler attacks, airstrikes, raids, extrajudicial assassinations, missile exchanges. Gaza, West Bank, Lebanon, Iran, Yemen as relevant. Civilian casualty counts, displacement events.
- **Legal/institutional machinations** — Knesset legislation, settlement expansion, judicial changes affecting occupied territories, IDF policy changes, ICC/ICJ developments.
- **Key-figure rhetoric** — Netanyahu, Smotrich, Ben-Gvir, Gallant (or successor), Halevi, opposition figures (Lapid, Gantz), settler leadership, IDF spokespersons. Also non-Israeli figures of prestige: US politicians, European foreign ministers, religious leaders, academics, athletes/celebrities making substantive statements.
- **International policy threads** — AIPAC influence, ICC arrest warrants, BDS policy fights, US weapons transfers, regional diplomatic shifts.

Out of scope: generic US domestic politics not connected to Israel. Generic Mideast news without an Israel/Palestine connection.

---

## File model

One `.md` file per story. Filename: `YYYY-MM-DD-slug.md` where slug is kebab-case, max ~6 words.

Examples:
- `2026-04-30-smotrich-jenin-operation.md`
- `2026-04-30-icc-warrant-amended.md`
- `2026-04-29-settler-attack-huwara.md`

Multiple files per day are expected. The viewer's Archive dropdown shows them reverse-chronologically.

---

## Registry files

| File | Purpose |
|------|---------|
| `sources.json` | Trusted source list — scan method, trust tier, topic relevance |
| `_threads.json` | Thread index — ongoing story arcs with entry lists |
| `_actors.json` | Actor slug registry — ensures consistent spellings across files |
| `_frontpage.json` | Routine's daily frontpage recommendation (lead, secondaries, promoted) |
| `_promotion-log.json` | Promotion history for slow-cycle cool-down enforcement |
| `index.json` | Flat index of all entries, regenerated by `scripts/rebuild-indexes.py` |
